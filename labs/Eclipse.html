Laboratory: Create a Project

Summary: In this laboratory, you will create your first project in Eclipse

Exercises

Exercise 0: Prerequisites

Make sure you’ve completed and understood the reading Introducing the IDE before starting this lab.

Exercise 1: Your First Program

a. Make a new project in Eclipse and call it HelloEclipse (File > New > Project)

b. Create a package in the class that will hold your class files (File > New > Package)

c. Create a class in the package called HelloJava (File > New > Class)

d. Copy and paste this code into your class to see it come to life (make sure you change the package name to what you’ve named yours) 

package first;

public class HelloJava {

   public static void main(String[] args) {
       System.out.print("I am the 1337 h4x0r. Phear me!");
   } // main(String[] args)
} // class HelloJava

e. Run the program and marvel at the words that appear in the console!

Exercise 2: Your Second Program

a. Make another class in your package and call it SimpleMath

b. Using the skeleton from above write a program that prints out the result of two integers and tells you which integers it print. An example output could be: Adding 3 and 4 gives us 7.

c. Java’s System.out.println() method will add them for you if you don’t put the numbers in the parenthesis as strings.

d. In Java you can concatenate string by putting a + sign between them like so: “Java” + “ ” + “is” + “ “ + “awesome”

e. Can you predict what happens when you put this in the print statement and run it?

f. There’s many more ways to print in Java, but a similar to the one you’ve already seen is System.out.print(). What’s the difference between the two? To make sure that you’ve got the answer create several print statements. Some with System.out.print() and some with System.out.println()
